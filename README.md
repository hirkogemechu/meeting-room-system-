## 📋 Overview

A **production-grade Meeting Room Management System** built with Node.js, Express, and Prisma. This enterprise solution allows organizations to efficiently manage meeting rooms, equipment, and bookings with real-time conflict detection and role-based access control.

### ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **Authentication** | JWT-based auth with refresh tokens & secure cookies |
| 👥 **Role-Based Access** | Separate dashboards for ADMIN and USER roles |
| 🏠 **Room Management** | Full CRUD operations with equipment tracking |
| 📅 **Smart Booking** | Automatic conflict detection prevents double bookings |
| 📧 **Email Notifications** | Booking confirmations and reminders |
| 📄 **PDF Receipts** | Generate booking receipts with worker threads |
| 📊 **Data Export** | Export bookings as CSV/JSON for reporting |
| 🎨 **Modern UI** | Professional, responsive interface inspired by DeskFlex |
| 🛡️ **Security** | Password hashing, JWT, rate limiting, Helmet.js |
